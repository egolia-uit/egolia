### Get transactions

- **Method:** `GET`
- **Path:** `/billing/transactions`
- **Tags:** Billing

#### Parameters

##### `page`

- **In:** `query`

Page number for pagination

`integer`, default: `1`

##### `limit`

- **In:** `query`

Number of items per page

`integer`, default: `20`

##### `order`

- **In:** `query`

Sort order

`string`, possible values: `"asc", "desc"`, default: `"desc"`

##### `courseId`

- **In:** `query`

`string`, format: `uuid`

##### `learnerId`

- **In:** `query`

If requested by non admin, then it will be that user

`string`, format: `number` — User ID from Authentik (need to change subject mode to User's ID instead of hashed)

#### Responses

##### Status: 200 Learner billing history

###### Content-Type: application/json

- **`data` (required)**

  `array`

  **Items:**

  - **`amount` (required)**

    `integer`, format: `int64`

  - **`courseId` (required)**

    `string`, format: `uuid`

  - **`createdAt` (required)**

    `string`, format: `date-time`

  - **`id` (required)**

    `string`, format: `uuid`

  - **`status` (required)**

    `string`, possible values: `"pending", "completed", "failed"` — Current status of a billing transaction

  - **`updatedAt` (required)**

    `string`, format: `date-time`

  - **`userId` (required)**

    `string`, format: `number` — User ID from Authentik (need to change subject mode to User's ID instead of hashed)

- **`pagination` (required)**

  `object`

  - **`hasNext` (required)**

    `boolean` — Whether there is a next page

  - **`hasPrev` (required)**

    `boolean` — Whether there is a previous page

  - **`limit` (required)**

    `integer` — Number of items per page

  - **`page` (required)**

    `integer` — Current page number

  - **`total` (required)**

    `integer` — Total number of items

  - **`totalPages` (required)**

    `integer` — Total number of pages

**Example:**

```json
{
  "data": [
    {
      "id": "4c7c8f0a-4f6e-4da1-867d-38f1120f0e5d",
      "userId": null,
      "courseId": "0f4f19b1-1602-4c12-ac1a-9ad5baf1e4dc",
      "amount": 499000,
      "status": null,
      "createdAt": "2026-03-25T08:30:00Z",
      "updatedAt": "2026-03-25T08:30:00Z"
    }
  ],
  "pagination": {
    "page": 1,
    "limit": 20,
    "total": 100,
    "totalPages": 5,
    "hasNext": true,
    "hasPrev": false
  }
}
```
