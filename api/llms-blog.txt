# Egolia API

- **OpenAPI Version:** `3.1.1`
- **API Version:** `0.0.0`

Egolia API

## Servers

- **URL:** `http://api.egolia.localhost`

## Operations

### Search posts

- **Method:** `GET`
- **Path:** `/blog/posts`
- **Tags:** Post

#### Parameters

##### `q`

- **In:** `query`

Full-text keyword for post search

`string`

##### `tag`

- **In:** `query`

Filter posts by tag

`string`

##### `page`

- **In:** `query`

Page number for pagination

`integer`, default: `1`

##### `limit`

- **In:** `query`

Number of items per page

`integer`, default: `20`

##### `order`

- **In:** `query`

Sort order

`string`, possible values: `"asc", "desc"`, default: `"desc"`

#### Responses

##### Status: 200 Matched blog posts

###### Content-Type: application/json

- **`data` (required)**

  `array`

  **Items:**

  - **`authorId` (required)**

    `string`, format: `number` — User ID from Authentik (need to change subject mode to User's ID instead of hashed)

  - **`commentCount` (required)**

    `integer`

  - **`content` (required)**

    `string`

  - **`createdAt` (required)**

    `string`, format: `date-time`

  - **`id` (required)**

    `string`, format: `uuid`

  - **`tags` (required)**

    `array`

    **Items:**

    `string`

  - **`title` (required)**

    `string`

- **`pagination` (required)**

  `object`

  - **`hasNext` (required)**

    `boolean` — Whether there is a next page

  - **`hasPrev` (required)**

    `boolean` — Whether there is a previous page

  - **`limit` (required)**

    `integer` — Number of items per page

  - **`page` (required)**

    `integer` — Current page number

  - **`total` (required)**

    `integer` — Total number of items

  - **`totalPages` (required)**

    `integer` — Total number of pages

**Example:**

```json
{
  "data": [
    {
      "id": "74af02ce-4e69-4ca1-a721-c0e19c9937b8",
      "authorId": null,
      "title": "Cach toi toi uu hanh trinh hoc tap",
      "content": "Day la noi dung bai viet chia se kinh nghiem hoc tap tren Egolia.",
      "tags": [
        "learning",
        "productivity"
      ],
      "createdAt": "2026-03-25T08:00:00Z",
      "commentCount": 12
    }
  ],
  "pagination": {
    "page": 1,
    "limit": 20,
    "total": 100,
    "totalPages": 5,
    "hasNext": true,
    "hasPrev": false
  }
}
```

##### Status: 401 Unauthorized Error response

###### Content-Type: application/json

**Example:**

```json
{}
```

###### Content-Type: application/json
